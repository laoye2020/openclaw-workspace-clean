# 📱 小米手机日历配置指南

> 把打卡提醒同步到手机闹钟

---

## 第一步：同步日历文件到手机

### 方法A：通过邮件（推荐）
1. 我会把 `calendar.ics` 文件发到你邮箱
2. 手机打开邮件，点击附件
3. 选择"用日历打开"
4. 导入完成！

### 方法B：通过小米云盘
1. 把文件上传到小米云盘
2. 手机下载后点击导入

### 方法C：USB连接
1. 把文件拷到手机 Download 文件夹
2. 文件管理器中找到点击导入

---

## 第二步：小米手机日历设置

### 1. 打开日历APP
- 找到"老爷打卡系统"日历
- 点击进入

### 2. 设置提醒方式（关键！）

**路径：日历 > 设置 > 提醒设置**

```
✅ 开启"默认提醒"
✅ 提醒方式选择："闹钟"（不是通知！）
✅ 提前提醒时间：0分钟（准时响）
✅ 铃声：选择刺耳的闹钟铃声
✅ 振动：开启
```

### 3. 关闭静音模式拦截

**路径：设置 > 声音与振动 > 勿扰模式**

```
⚠️ 确保日历APP不在勿扰拦截列表
或者：设置日历为例外应用
```

### 4. 允许后台运行（防杀）

**路径：设置 > 应用设置 > 日历 > 省电策略**

```
选择："无限制"
（防止系统清理后台导致不提醒）
```

---

## 第三步：测试提醒

### 创建一个测试事件
1. 在日历新建一个事件
2. 设置1分钟后提醒
3. 锁屏等待
4. 应该听到闹钟声并看到全屏提醒

### 如果测试成功
- 说明配置正确
- 明天的打卡提醒会准时响起

### 如果测试失败
检查：
- [ ] 日历APP有通知权限吗？
- [ ] 提醒方式设为"闹钟"了吗？
- [ ] 手机静音了吗？
- [ ] 勿扰模式拦截了吗？
