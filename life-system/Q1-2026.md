# 🚀 Q1 2026 战役计划

> 时间：2026年2月 - 4月
> 主题：**系统启动月**

---

## 🎯 Q1 核心目标

### 1. 健身：建立训练习惯
- 每周4练，12周完成48次训练
- 学会基础动作，避免受伤
- 体重维持135斤，体脂开始下降

### 2. 财富：评估现状 + 开源节流
- 盘点当前资产和收入
- 制定主业提升计划
- 确定副业方向

### 3. 被动收入：确定方向
- 研究至少3个被动收入模式
- 选择1个开始尝试
- 完成基础技能学习

---

## 📅 Q1 月度拆解

### 2月：启动月

**健身**：
- [ ] 办健身卡或购买家用器械
- [ ] 学习基础动作（深蹲、卧推、硬拉、推举）
- [ ] 建立训练日志
- [ ] 目标：每周3-4练，完成12次训练

**财富**：
- [ ] 盘点当前财务状况（收入、支出、资产、负债）
- [ ] 建立记账习惯
- [ ] 研究主业提升路径
- [ ] 初步调研副业方向

**被动收入**：
- [ ] 研究：自媒体/软件开发/电商/投资
- [ ] 确定1个主攻方向
- [ ] 购买/学习相关课程或书籍

**系统**：
- [ ] 完善豆芽AI工作流
- [ ] 建立每周日复盘习惯
- [ ] 阅读2本书

---

### 3月：执行月

**健身**：
- [ ] 增加训练强度
- [ ] 学习饮食计算（蛋白质、碳水、脂肪）
- [ ] 目标：每周4练，完成16次训练

**财富**：
- [ ] 执行主业提升计划
- [ ] 开始副业第一个项目
- [ ] 储蓄率达到50%

**被动收入**：
- [ ] 完成第一个MVP
- [ ] 发布/上线
- [ ] 收集反馈

**系统**：
- [ ] 第一个月度复盘
- [ ] 阅读2本书

---

### 4月：优化月

**健身**：
- [ ] 评估2个月成果
- [ ] 调整训练计划
- [ ] 目标：每周4练，完成16次训练

**财富**：
- [ ] Q1财务复盘
- [ ] 调整Q2计划

**被动收入**：
- [ ] 优化第一个项目
- [ ] 争取产生第一笔收入

**系统**：
- [ ] Q1整体复盘
- [ ] 制定Q2计划
- [ ] 阅读2本书
